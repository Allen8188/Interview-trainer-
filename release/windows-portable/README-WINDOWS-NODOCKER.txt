Interview Trainer - Windows Portable (No Docker)

Quick Start:
1) Double-click:
   Launch-Interview-Trainer-NoDocker.bat
2) Wait 2-5 seconds.
3) Open in browser:
   http://127.0.0.1:8000

Stop:
- Double-click:
  Stop-Interview-Trainer-NoDocker.bat

Notes:
- This package does NOT require Docker.
- Python is bundled inside the executable build.
- If Windows Firewall prompts, allow access on Private network.
