Interview Trainer - Windows Quick Start

1) Install Docker Desktop for Windows:
   https://www.docker.com/products/docker-desktop/

2) Double-click:
   Launch-Interview-Trainer.bat

3) Wait until you see:
   Interview Trainer started at http://localhost:8000

4) Browser should open automatically. If not, open:
   http://localhost:8000

Stop:
 - Double-click Stop-Interview-Trainer.bat

Notes:
 - This app runs in Docker containers (no Python local install required).
 - If Windows Firewall prompts, allow Docker access on Private network.
